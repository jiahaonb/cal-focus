# Cal Focus 项目说明

## 项目概述

- 项目名称：Cal Focus
- 技术栈：C# + WinUI 3 + Windows App SDK + SQLite + JSON
- 目标：构建面向 Windows 桌面的日历日程、提醒计划与桌面小组件效率工具

---

## 当前功能状态

### 1. 主界面与导航
- 已实现左侧导航与页面切换。
- 已实现主窗口 2.1:1 等比缩放约束，默认尺寸 1260x600。
- 已实现导航状态持久化。

### 2. 首页（Schedule）
- 固定三栏布局，页面无滚动条。
- 左栏：时间日期、天气占位（预留 API）、未来提醒。
- 中栏：未来日程列表（内部滚动）。
- 右栏：未来计划/待办列表（内部滚动）。
- 三类列表统一为紧凑长条样式：
  - 第一行：事件标题
  - 第二行：小字时间-日期

### 3. 日历日程（CalendarSchedule）
- 默认仅显示完整月历网格。
- 单击日期仅高亮主题边框；今天样式常驻显示且不与选中态混用。
- 已修复双击命中问题，避免单击事件吞掉双击手势。
- 双击日期数字弹出当日详情卡片（含日程/提醒/待办三块信息）。
- 双击日期空白区仅打开“新建日程”卡片。
- 头部新增“详情”按钮：可将选中日期的详情固定在右侧面板展示。
- 右侧详情展开时自动扩展主窗口宽度，避免压缩日历网格。
- 月历单元采用紧凑样式，避免日历项显示不全。

### 4. 待办与提醒
- 已实现 Todo / Reminder 数据服务。
- 已实现本地持久化（productivity.data.json）。
- 已实现提醒到时通知触发。

### 5. 小组件
- 已支持时钟、日程、番茄钟组件创建与管理。
- 小组件中心已改为静态预览卡片式添加入口。
- 三类小组件已统一为 iOS 风格深色玻璃化视觉（层叠圆角、柔和高光、半透明材质）。
- 小组件窗口已移除传统标题栏/边框外观。
- 小组件新增右上角三点菜单，支持调整大小、关闭组件、修改组件颜色。
- 新建小组件默认采用当前主题色，同时支持每个小组件独立改色并持久化到布局文件。
- 番茄钟组件移除 `-1/+1` 快捷按钮，保留分钟输入与开始/暂停/重置操作。

### 6. 帮助关于页（HelpAbout）
- 已修复内容右偏和加载不完整问题，页面改为居中布局并禁用横向滚动溢出。

### 7. 主题系统
- 已实现全局 DesignSystem 资源字典。
- 已实现主题设置服务（品牌色、玻璃、动效与模糊设置）。
- 首页与日历页的关键视觉元素由主题资源驱动。
