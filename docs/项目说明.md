# Cal Focus 项目说明

## 📋 项目概述

**项目名称**: Cal Focus - 日历提醒计划与桌面小组件  
**技术栈**: C# + WinUI 3 + Windows App SDK + SQLite  
**目标**: 打造一款以日历日程 + 提醒 + 计划管理 + 桌面小组件为核心的 Windows 桌面软件

---

## 🏗️ 项目架构

### 分层结构
```
CalFocus.App/              # 启动与 UI 层
CalFocus.Core/             # 业务实体、用例、规则
CalFocus.Infrastructure/   # SQLite、文件、系统 API 适配
CalFocus.Widgets/          # 小组件窗口与渲染
CalFocus.Theming/          # 主题、材质、视觉 token
```

### 核心模块
- **日程管理** - CRUD + 重复规则引擎
- **提醒系统** - 提醒 CRUD + Windows 通知
- **计划表** - 周/月视图 + 状态流转
- **小组件系统** - 桌面时钟、日程小组件
- **多显示器** - 自动检测、跨屏拖动、布局持久化
- **主题系统** - Glass/Flat/Depth 风格切换

## 🔑 关键技术

### 数据持久化
- **SQLite** - 业务数据存储
- **JSON** - 配置和布局存储
- **Repository 模式** - 数据访问抽象

### 业务逻辑
- **值对象模式** - RecurrenceRule 不可变设计
- **Service 模式** - 业务逻辑封装
- **事件驱动** - UI 更新通知机制

### UI 框架
- **WinUI 3** - 现代 Windows UI
- **XAML** - 声明式 UI 定义
- **DispatcherTimer** - 实时更新

### 数据层
- `CalFocus.Infrastructure/Persistence/Sqlite/ScheduleRepository.cs` - 日程数据访问
