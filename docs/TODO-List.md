## 最近更新（2026-04-01）

### 首页（Schedule）
- 改为固定三栏布局，页面级无滚动条。
- 左栏：时间日期、天气预留、未来提醒。
- 中栏：未来日程（内部滚动）。
- 右栏：未来计划/待办（内部滚动）。
- 三类列表统一为紧凑长条卡片：第一行标题，第二行小字时间日期。

### 日历日程（CalendarSchedule）
- 月历单元改为“单击选中边框”与“今天常驻高亮”分离，避免样式重叠。
- 修复双击命中问题：双击数字稳定弹出详情卡片，双击空白稳定打开新建日程卡片。
- 双击日期数字弹出详情卡片（包含日程/提醒/待办三块）。
- 双击日期空白区仅弹出“新建日程”卡片。
- 头部新增“详情”按钮：单击选中某天后可在右侧展开详情面板。
- 右侧详情展开时会优先扩展主窗口宽度，避免挤压日历区域。

### 桌面小组件（Widgets）
- 时钟、日程、番茄钟统一升级为深色玻璃质感卡片（多层圆角、柔和高光、半透明材质）。
- 小组件窗口改为无传统标题栏/边框外观，减少 Windows 旧式观感。
- 每个小组件新增右上角三点菜单：可调整大小、关闭组件、修改组件颜色。
- 新建小组件默认跟随当前主题色，同时支持对单个小组件独立改色并持久化。
- 番茄钟去除 `-1/+1` 调整按钮，仅保留分钟输入、开始/暂停、重置。

### 帮助关于（HelpAbout）
- 修复页面内容右偏与显示不完整问题，改为居中容器并禁用横向溢出滚动。

### 主题联动
- 首页与日历页的标题、边框、强调色改为主题资源驱动。
- 主题颜色变更可影响页面关键视觉元素，不再仅影响单一新增卡片。

### 文档
- 清理旧的错误进度描述与过期“进行中”状态。
- 对应需求同步到当前实现状态与后续待办。
